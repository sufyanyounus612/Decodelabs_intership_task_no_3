# SQL Data Analytics Project using Python

A professional Data Analytics project built using Python, Pandas, and SQLite to perform SQL-based data analysis on real-world business datasets. This project demonstrates how structured SQL queries can be integrated with Python to extract actionable insights, analyze business performance, and generate meaningful reports efficiently.

## Key Features

* Data preprocessing and cleaning using Pandas
* SQL query execution using SQLite
* Data filtering with `WHERE` clause
* Data sorting using `ORDER BY`
* Grouped analysis with `GROUP BY`
* Aggregation operations using:

  * `COUNT()`
  * `SUM()`
  * `AVG()`
* Revenue and sales performance analysis
* Order and customer analytics
* Professional, modular, and well-structured Python code

## Technologies Used

* Python
* Pandas
* SQLite3
* Jupyter Notebook
* VS Code

## Project Objectives

The primary objective of this project is to demonstrate how SQL can be combined with Python for efficient business data analysis and reporting. The project focuses on transforming raw datasets into meaningful insights through structured querying techniques.

### Business Insights Generated

* Revenue analysis
* Product performance tracking
* Order status reporting
* Customer referral source analysis
* Payment method performance evaluation
